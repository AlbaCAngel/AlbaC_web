Donde se inserta Javascript

  • debe ser insertado en las etiquetas <script>
  • Puede ir en el <head> o en el <body>,
  • Pueden ser colocados en archivos externos js/funciones.js
  • El código JavaScript puede ser insertado en líneas, en las etiquetas HTML dentro del atributo de evento onClick

Ejemplo: Javascript en el Head


<head>
<script>
function myFunction() {
  document.getElementById("demo").innerHTML = "Paragraph changed.";
}
</script>
</head>
