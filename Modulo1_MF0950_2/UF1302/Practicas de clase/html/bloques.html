Etiquetas que citan el texto

las abreviaturas de marcado <abbr> pueden prooporcionar informaciín útil a la los navegadores, sistemas de traducción y motores de búsqueda. Utiliza el atributo <title> para definir el significado de la abreviación.

La etiqueta de marcado <cita> hace referencia a un trabajo creativo como un libro, una película, un programa de televisión o una canción, no el nombre de una persona.

La etiqueta de marcado <blockquote> define una cita larga que se representa como un elemento de bloque. No genera automáticamente comillas alrededor del contenido.

La etiqueta de marcado <q> define una cita corta en la línea. Mrca las citas a otros autores o documentos. Genera automáticamente comillas alrededor del contenido.

La etiqueta de marcado <dfn> define un término y debe contener el atributo <title> para la definición/explicación del término de dentro.

Elementos que agrupan el texto

El elemento<div> se utiliza a menudo como un contenedor para otros elementos HTML. El elemento <div>tiene atributos pero no son necesarios, los más comunes son style, class y id. para dar estilos al CSS, al bloque de contenido.

El elemento <span> se utiliza como un contenedor para agrupar un texto en línea. El elemento <span>, también tiene atrivutos, pero tampoco son necesarios, los más comunes son style, class y id. para dar estilos al CSS, al bloque de contenido.
