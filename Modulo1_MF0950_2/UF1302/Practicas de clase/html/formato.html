Formato HTML

El formato de texto, es el marcado que se aplica a un texto para añadirle datos a la forma

Al texto resultante sele conoce como texto formateado, texto con estilos o texto enriquecido

Encabezados de texto

Los motores de búsqueda utilizan los encabezados para indexar la estructura y el contenido de sus páginas web.

Los usuarios hojear sus páginas por sus títulos. Es importante utilizar las partidas para mostrar la estructura del documento.

Los encabezados <h1>deben ser utilizados para los títulos principales, seguido por <h2> menos importantes <h3>.

Nota: Usar encabezados HTML para sólo encabezados. No utilice encabezados para que el texto GRANDE o negrita.

Etiquetas que dan formato al texto

HTML también define especiales elementos para la definición de texto con un especial significado .

HTML utiliza elementos como <b>y<i>para el formato de salida, como negrita o cursiva texto.

Etiquetas que separan el texto

La etiqueta <p> separa el texto, añadiendo automaticamente un espacio(margen)antes y después de cada elemento &tl;p>. El navegador eliminará todos los espacios en blanco yñas líneas adicionales cuando se visualiza la etiqueta <p>.

la etiqueta <br> es una etiqueta vacía, lo que significa que no tiene ninguna etiqueta final. Inserta un salto de línea (una nueva línea) sin comenzar un nuevo párrafo.

Un texto dentro de una etiqueta <pre> se muestra en una fuente de ancho fijo (generalmente Courier), y conserva ambos espacios y saltos de línea.

Nota: Utilice el elemento <pre>cuando quiera mostrar el texto con formato inusual, o algun tipo de código informático.




Nota: Los navegadores muestran <strong>como <b>, y <em>como <i>. Sin embargo, hay una diferencia en el significado de las siguientes etiquetas: <b>y <i>define el texto en negrita y cursiva, pero <strong>y <em>significa que el texto es "importante".
