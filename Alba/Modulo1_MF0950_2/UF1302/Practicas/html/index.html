Introdución HTML

What is HTML?

HTML is the standard markup language for creating Web pages.
  • HTML significa Hyper Text Markup Language
  • HTML describe la estructura de las páginas Web usando marcado
  • elementos HTML son los bloques de construcción de páginas HTML
  • elementos HTML están representados por etiquetas
  • HTML etiquetas de la etiqueta de piezas de contenido como "mesa" "la partida", "párrafo" , y así sucesivamente
  • Los navegadores no muestran las etiquetas HTML, pero las usan para representar el contenido de la página

Estructura básica de un documento HTML

<!DOCTYPE html>
<html lang="es-ES">

<head>
   <title>Page Title</title>
   <meta charset="UTF-8">
</head>

<body>

  <h1>My First Heading</h1>
  <p>My first paragraph.</p>

</body>
</html>

Explicación de etiquetas básicas

  1. La etiqueta <!DOCTYPE html> declaración define el tipo documento para ser HTML5
  2. La etiqueta <html> define el elemento raíz de una página HTML
  3. El atributo lang="es-ES" especifica el idioma del documento, va seguido de la etiqueta <html>
  4. La etiqueta <head> define los metadatos que contiene información sobre el documento
  5. La etiqueta <meta charset="UTF-8"> define el estándar de codificación universal de caracteres, va dentro de head.
  6. La etiqueta <title> especifica un título para el documento
  7. La etiqueta <body> define el contenido visible de la página
  8. La etiqueta<h1> define un encabezado del texto
  9. La etiqueta <p> define un párrafo

Etiquetas HTML

Las etiquetas HTML son los nombres de ekementos entre paréntesis angulares


<Nombre de etiqueta> contenido va aquí ... </etiqueta>


Etiquetas HTML
Normalmente vienen en pares como <p>y</p>
La primera etiqueta
En un par es la etiqueta de inicio, la segunda etiqueta es la etiqueta de fin
La etiqueta final
Se escribe igual que la etiqueta de inicio, pero con una barra inclinada insertada antes de que el nombre de la etiqueta

Consejo: La etiqueta de inicio también se llama la etiqueta de apertura y la etiqueta final de la etiqueta de cierre.
