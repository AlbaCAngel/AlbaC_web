Formato HTML

El formato de texto, es el marcado que se aplica a un texto para añadirle datos a la forma

Al texto resultante sele conoce como texto formateado, texto con estilos o texto enriquecido

Encabezados de texto

Los motores de búsqueda utilizan los encabezados para indexar la estructura y el contenido de sus páginas web.

Los usuarios hojear sus páginas por sus títulos. Es importante utilizar las partidas para mostrar la estructura del documento.

Los encabezados <h1>deben ser utilizados para los títulos principales, seguido por <h2> menos importantes <h3>.

Nota: Usar encabezados HTML para sólo encabezados. No utilice encabezados para que el texto GRANDE o negrita .

Etiquetas que dan formato al texto

HTML también define especiales elementos para la definición de texto con un especial significado .

HTML utiliza elementos como <b>y<i>para el formato de salida, como negrita o cursiva texto.

Etiqueta Definición Ejemplo Utilización
<b> Define un texto en negrita Este texto está en negrita <b> Este texto está en negrita </b>
<i> Define un texto en cursiva Este texto es en cursiva <i> Este texto está en cursiva </i>
<strong> Define un texto importante Este texto es importante <strong> Este texto es importante </strong>
<em> Define un texto enfatizado Este texto está enfatizado <em> Este texto está enfatizado</em>
<small> texto pequeño Este texto es mas pequeño Este texto es <small> mas pequeño</small>
<sup> Define un texto en superíndice Este texto es superíndicedel texto Este texto es <sup> superíndice</sup>de el texto
<sub> Define un texto en subíndice Este texto es subíndice del texto Este texto es un <sub> subíndice</sub> del texto
<ins> texto insertado Este texto es insertado (añadido) después Este texto esinsertado <ins> (añadido)</ins> después
<s> Define un texto tachado Este texto se ha tachado Este texto se ha <s> tachado</s>
<u> Marca o subraya un texto Este texto está subrayado Este texto está <u> subrayado </u>
<del> Simboliza un texto borrado Este texto se ha eliminado Este texto se ha <del> eliminado</del>
